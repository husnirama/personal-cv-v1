# Profesional CV Template

## Features :

1. VIEW MORE - VIEW LESS BUTTON ON PORTFOLIO PAGE
2. DOWNLOAD PAGE
3. LINKED TO WEB WHATSAPP ON HIRE ME
4. DIRECT SEND EMAIL

## Package Use

### - Vite

Installation command : `npm create vite@latest my-vite-app`

### - Tailwindcss

Installation command `npm install tailwindcss @tailwindcss/vite`
Configuration : https://tailwindcss.com/docs/installation/using-vite

### - react-router-dom

Installation command : `npm install react-router-dom`
